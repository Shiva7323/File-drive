About File Drive

Your modern file management and collaboration platform

Our Mission

File Drive is designed to simplify file management and team collaboration. We believe that organizing and sharing files should be effortless, secure, and intuitive.

Whether you're working solo or with a team, File Drive provides the tools you need to store, organize, and collaborate on your files effectively.

Key Features

Single Mode

Personal file storage with complete privacy. Perfect for individual users who want to keep their files organized and accessible.

Team Mode

Collaborative workspace with role-based permissions. Ideal for teams that need to share and work on files together.

Team Chat

Real-time messaging with message editing and deletion capabilities. Stay connected with your team members.

File Editing

Edit text files directly in your browser. Support for various file types including code files and documents.

Secure Storage

Your files are stored securely with proper access controls and user authentication.

Responsive Design

Access your files from any device with our mobile-friendly, responsive interface.

Technology Stack

Backend
  • Flask - Web framework
  • SQLAlchemy - Database ORM
  • SQLite - Database
  • Werkzeug - Security utilities
Frontend
  • HTML5 & CSS3
  • Bootstrap 5 - UI framework
  • JavaScript - Interactivity
  • Font Awesome - Icons

Security & Privacy

User Security
  • Secure password hashing with scrypt
  • Session-based authentication
  • Role-based access control
  • Input validation and sanitization
File Security
  • Secure file upload handling
  • File type validation
  • Permission-based file access
  • Soft delete for data recovery
