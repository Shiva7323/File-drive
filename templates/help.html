Help & Guidelines

Everything you need to know about File Drive

Getting Started

1. Create Account

Sign up with a unique username and secure password. Username must be 3-50 characters with letters, numbers, and underscores only.

2. Choose Mode

Switch between Single Mode (personal files) and Team Mode (collaborative workspace) based on your needs.

Understanding Modes

Single Mode
  • Personal file storage
  • No team requirements
  • Private workspace
  • Direct file management
Team Mode
  • Collaborative workspace
  • Team permissions (Admin, Editor, Viewer)
  • Team chat functionality
  • Shared file management

File Management

Supported File Types
Text Files (Editable):
  • .txt - Plain text files
  • .md - Markdown files
  • .py - Python scripts
  • .js - JavaScript files
  • .html - HTML documents
  • .css - Stylesheets
  • .json - JSON data
  • .xml - XML documents
All File Types:
  • Images (.jpg, .png, .gif, etc.)
  • Documents (.pdf, .doc, .docx)
  • Archives (.zip, .rar, .7z)
  • Videos (.mp4, .avi, .mov)
  • Audio (.mp3, .wav, .flac)
  • And many more...

Team Features

Admin Role
  • Manage team settings
  • Delete any messages
  • Manage team members
  • Full file permissions
Editor Role
  • Upload and edit files
  • Send messages
  • Edit own messages
  • Browse team files
Viewer Role
  • Browse team files
  • Send messages
  • Download files
  • Read-only access

Chat Guidelines

Message Management
  • Edit: You can edit your own messages within 5 minutes of sending
  • Delete: You can delete your own messages anytime
  • Admin Delete: Admins can delete any message in the team
  • Formatting: Use line breaks for better message formatting

Security & Best Practices

Password Security
  • Use at least 8 characters
  • Include letters, numbers, and special characters
  • Don't share your password
  • Use unique passwords for different accounts
File Security
  • Only upload files you own or have permission to share
  • Be careful with sensitive information in team chats
  • Regularly review team member permissions
  • Report suspicious activity to team admins

Troubleshooting

Common Issues
  • Can't upload: Check file size and type restrictions
  • Can't edit: Only text files are editable online
  • Chat not working: Refresh the page or check team membership
  • Permission denied: Contact your team admin
Quick Fixes
  • Clear browser cache and cookies
  • Try a different browser
  • Check your internet connection
  • Log out and log back in
